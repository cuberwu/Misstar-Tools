工具箱简介：
MT工具箱是目前第三方插件里面最为方便易用的插件集合，往往我们以前安装各种插件，都要一个个在SSH里面安装并且还一个个设置都是比较麻烦的！MT工具箱给我们提供一个整体的框架，在工具箱里面的每个插件都是独立的。所以我们可以自由选择需要自己安装的插件，并且可以单独更新插件。MT工具箱直接集成在路由器后台页面，只有安装的时候需要一行SSH命令，往后的插件安装卸载升级都只需要在这一个后台页面进行，全图形化操作，特别的方便！有问题的时候不用再经常使用SSH去查看每个插件的运行状况和设置了！并且工具箱是免疫OTA升级的，可以放心更新系统，工具箱依然坚挺的存在！
 

插件介绍：
MT工具箱目前提供的插件还是比较丰富的，基本能满足需求。包括如下插件：KMS服务器，VSFTP服务器，**服务器，远程管理，ARIA2，Koolproxy广告过滤，阿呆喵广告过滤，webshell，aliddns，frp内网穿透。
当然后续会提供更多的插件，直接在页面的插件中心安装即可。
 
**服务器：
这个插件可以让路由建立一个**服务器，用于外网连接路由器，可以用于读取路由内网共享资源，一些只有内网能操作的东西，可以联机玩游戏等等，这个可以自行搜索下**！由于R1D有些问题没有这个插件所以这里没有相关图片之类的提供！有兴趣的可以安装工具箱！

KMS服务器：
KMS大家应该都不陌生，没错，我们现在激活工具用的最多的就是KMS软件了！不过大家都这道，网上这类的工具比较乱，很多都附带恶意软件！这个插件是直接把路由器变成一个KMS服务器，支持内网和外网（配合DDNS）使用，十分方便！以后激活就靠自己就好了！
 

VSFTP服务器：
之前论坛上有朋友发过VSFTPD这个插件了，本人也转载过一份，但是安装和配置起来都比较麻烦！现在MT工具箱把这个工具图形化了，设置很方便，同时也支持多用户登录不同文件夹，满足对于一个基本FTP服务器的需求！配合DDNS还可以开启外网访问，算是一个简易云存储了！
 

远程管理&webshell：
这个插件主要是提供远程访问的端口设置，支持远程SSH登录和web管理后台登录。方便远程设置的操作需求
webshell则是提供一个网页端的SSH终端，
  

广告过滤（ADM广告过滤&koolproxy广告过滤）：
MT工具箱提供了两款广告过滤工具，ADM很多人应该都不陌生了，但是路由版的ADM效果一般并且十分耗费资源所以不推荐使用。好在最近作者添加了KP广告过滤插件，KP也很出名，移植自梅林固件同名插件，去广告的效果十分赞，占用资源也低，还支持每台机器单独配置，支持https，主程序和规则也可以自动更新，个人十分推荐这个插件！
  


ARIA2：
下载神器ARIA2也被搬上来了，作为一个路由上第二个下载工具还是不错的，特别是百度云，你懂的！当然，也是支持远程的（配合DDNS使用）
 

Aliddns：
阿里ddns工具，方便域名在阿里的同学绑定解析。个人使用下来阿里比固件自带的花生壳要靠谱一点，速度和稳定性方面！不过阿里没有免费的域名提供，如果没有域名在阿里则需要注册一个！好在win域名一个一年才4块钱！

由于不可描述的原因，已做下架处理。

FRP内网穿透：
大内网福音，支持内网穿透，端口反向代理，对于内网用户（比如移动宽带）这些用不了ddns的用户只能使用这类内网穿透插件了！同时也是支持端口反向代理，比如被封锁的80端口，当然，需要frp服务器支持。
